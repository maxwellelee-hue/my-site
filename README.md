# my-site

A personal website — currently a foundation, with more features planned.

Built with plain HTML, CSS, and JavaScript. No build tools or frameworks.

## How to open it

Double-click `index.html` to open in your browser. No installation needed.

After editing a file, save it and refresh the browser (F5) to see changes.

## Short-session workflow

Each time you sit down to work, end the session with these three commands:

```bash
git add -A
git commit -m "describe what you changed"
git push
```

That's the whole loop. Every push saves your work permanently to GitHub so nothing is ever lost between sessions.

## Files

- `index.html` — page structure
- `styles.css` — colors, fonts, layout
- `script.js` — interactive behavior
